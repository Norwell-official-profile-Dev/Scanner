QR & Barcode Scanner

Scanner Capabilities

This application supports scanning both QR codes and various barcode formats using your device's camera or uploaded images.

Supported Formats

  • QR Code: Scanned using the jsQR library, supports standard QR codes containing URLs, text, or other data.
  • Barcodes (via ZXing library):
    • EAN-13
    • EAN-8
    • UPC-A
    • UPC-E
    • Code 128
    • Code 39
    • Code 93
    • Codabar
    • ITF (Interleaved 2 of 5)
    • RSS-14
    • RSS-Expanded
    • Data Matrix
    • Aztec
    • PDF417

Unsupported Formats

  • Some proprietary or niche barcode formats not supported by ZXing.
  • Damaged or low-quality codes that cannot be decoded.
  • Non-standard QR code variants not compatible with jsQR.

How Images Are Processed

Camera Scanning: The application captures video frames from your device's camera. Each frame is drawn onto a canvas, and the image data is analyzed using jsQR (for QR codes) or ZXing (for barcodes). When a code is detected, its content is displayed, and the device vibrates briefly.

File Upload: Uploaded images are loaded onto a canvas, and the image data is processed similarly using jsQR or ZXing, depending on the selected mode. If a code is detected, its content is displayed, and the device vibrates. If no code is found, an error message is shown.

Tips for Successful Scanning:

  • Ensure good lighting and focus.
  • Position the code centrally within the camera frame or image, aligning with the guiding lines.
  • Use high-resolution images for uploads.
  • Switch to the correct mode (QR or Barcode) for the code type.
